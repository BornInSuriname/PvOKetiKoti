# PvO Keti Koti Website

Deze website is ontwikkeld als headless platform voor de herdenking en educatie rondom Keti Koti, geïnitieerd door PvO (Platform van Organisaties).

De frontend draait op Next.js en haalt zijn content dynamisch op vanuit een gekoppelde WordPress CMS-installatie.

---

## 🌐 Live website
- **Publieke site:** [https://pvoketikoti.nl](https://pvoketikoti.nl)
- **CMS login:** [https://cms.pvoketikoti.nl/wp-admin](https://cms.pvoketikoti.nl/wp-admin)

---

## ⚙️ Technologieën
- **Next.js 14** – moderne frontend
- **Tailwind CSS** – voor design en styling
- **WordPress CMS** – headless contentbeheer
- **WPGraphQL + ACF** – content via GraphQL API
- **GraphQL Request** – frontend-data ophalen
- **Netlify** – hosting & CI/CD
